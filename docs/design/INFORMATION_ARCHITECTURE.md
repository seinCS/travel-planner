> **작성일**: 2026-01-26
> **버전**: 1.0

## 3. 사용자 플로우 다이어그램

### 3.1 핵심 플로우: 스크린샷 → 장소 추가

```
┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│ 스크린샷 │ → │  업로드  │ → │ AI 분석  │ → │ 장소 추출│ → │ 지도 표시│
│   저장   │    │  (입력탭)│    │  (로딩)  │    │  (목록)  │    │  (마커)  │
└──────────┘    └──────────┘    └──────────┘    └──────────┘    └──────────┘
                     │                               │
                     │         ┌──────────┐         │
                     └────────│  실패 시  │─────────┘
                              │ 실패 목록 │
                              │   표시   │
                              └──────────┘
```

### 3.2 일정 편성 플로우

```
┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│ 일정 생성│ → │ 날짜 선택│ → │ Day 생성 │ → │ 장소 배치│
│  (버튼) │    │ (캘린더) │    │  (자동)  │    │(드래그)  │
└──────────┘    └──────────┘    └──────────┘    └──────────┘
                                                     │
                                                     ▼
                              ┌──────────┐    ┌──────────┐
                              │ 시간 설정│ ← │ 순서 조정│
                              │  (선택)  │    │(드래그)  │
                              └──────────┘    └──────────┘
```

### 3.3 그룹 협업 플로우

```
┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│ 초대 링크│ → │ 링크 공유│ → │ 초대 수락│ → │ 멤버 추가│
│   생성   │    │ (카톡 등)│    │  페이지  │    │  완료   │
└──────────┘    └──────────┘    └──────────┘    └──────────┘
     │                               │
     │         ┌──────────┐         │
     └────────│ 로그인   │─────────┘
              │ 필요 시  │
              └──────────┘
```

### 3.4 AI 챗봇 장소 추가 플로우

```
┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│  챗봇   │ → │ 질문 입력│ → │ AI 응답 │ → │ 장소 추가│
│  열기   │    │ (자연어) │    │ (추천)  │    │  버튼   │
└──────────┘    └──────────┘    └──────────┘    └──────────┘
                                                     │
                                                     ▼
                              ┌──────────┐    ┌──────────┐
                              │ 일정에   │ ← │ 장소 목록│
                              │  추가   │    │  반영   │
                              └──────────┘    └──────────┘
```

## 4. 네비게이션 구조

### 4.1 글로벌 네비게이션

| 위치 | 요소 | 동작 |
|------|------|------|
| 좌측 상단 | Logo | 대시보드로 이동 |
| 우측 상단 | 프로필 | 드롭다운 (설정, 로그아웃) |

### 4.2 프로젝트 상세 네비게이션

| 위치 | 요소 | 동작 |
|------|------|------|
| 좌측 상단 | ← 화살표 | 대시보드로 이동 |
| 상단 중앙 | 탭 (장소/일정) | 탭 전환 |
| 우측 상단 | 멤버, 공유, 설정 | 모달/패널 열기 |
| 우측 하단 (고정) | 챗봇 버튼 | 챗봇 열기 |

### 4.3 모바일 네비게이션

| 위치 | 요소 | 설명 |
|------|------|------|
| 하단 고정 | 탭 바 | 지도/목록/입력/일정 전환 |

## 5. 상태별 화면 (States)

### 5.1 빈 상태 (Empty States)

| 화면 | 빈 상태 메시지 | CTA |
|------|---------------|-----|
| 대시보드 (프로젝트 없음) | "첫 여행을 계획해보세요" | [+ 새 프로젝트] |
| 장소 목록 (장소 없음) | "아직 저장된 장소가 없어요" | 이미지 업로드 안내 |
| 일정 (미생성) | "일정을 만들어보세요" | [일정 만들기] |
| 일정 Day (항목 없음) | "장소를 드래그해서 추가하세요" | - |

### 5.2 로딩 상태 (Loading States)

| 동작 | 로딩 표시 |
|------|----------|
| 이미지 업로드 | 프로그레스 바 + "업로드 중..." |
| AI 분석 | 스피너 + "AI가 장소를 찾고 있어요..." |
| 지도 로드 | 스켈레톤 + "지도 로딩 중..." |
| 데이터 로드 | 스켈레톤 UI |

### 5.3 에러 상태 (Error States)

| 상황 | 메시지 | 액션 |
|------|--------|------|
| 장소 추출 실패 | "장소를 찾지 못했어요" | [다시 시도] 또는 목록에 표시 |
| 네트워크 오류 | "연결에 문제가 있어요" | [다시 시도] |
| 권한 없음 | "접근 권한이 없어요" | 대시보드로 이동 |

## 6. 반응형 브레이크포인트

| 브레이크포인트 | 너비 | 레이아웃 |
|---------------|------|----------|
| **Mobile** | < 640px | 탭 네비게이션, 전체 화면 전환 |
| **Tablet (sm)** | 640-767px | 2-Column (지도 + 탭 사이드바) |
| **Tablet (md)** | 768-1023px | 2-Column (지도 + 탭 사이드바) |
| **Desktop (lg+)** | ≥ 1024px | 3-Column (지도 + 목록 + 입력) |
