## 1. 제품 개요

### 제품명
**Travel Planner** (트래블 플래너)

### 한 줄 소개
> SNS에서 발견한 장소를 실제 여행으로 연결하는 AI 기반 여행 플래너

### 제품 비전
"**SNS에서 여행으로**" - 인스타그램, 유튜브, X(트위터)에서 발견한 멋진 장소들을 저장하고, AI의 도움으로 실제 여행 일정으로 만들어가는 경험을 제공합니다.

### 미션
여행 계획의 가장 번거로운 과정인 "장소 수집 → 정리 → 일정 편성"을 자동화하여, 사용자가 여행의 설렘에만 집중할 수 있도록 합니다.

---

## 2. 타겟 사용자

### 주요 타겟: 20-30대 MZ세대 여행자

| 특성 | 설명 |
|------|------|
| **연령대** | 20-35세 |
| **디지털 리터러시** | SNS 활발 사용, 새로운 앱 적응 빠름 |
| **여행 스타일** | 자유여행 선호, 로컬 경험 중시 |
| **정보 수집** | 인스타그램, 유튜브, 블로그에서 여행 정보 획득 |
| **Pain Point** | SNS에서 본 장소를 따로 저장하고 정리하는 번거로움 |

### 사용자 특성
- SNS에서 여행 콘텐츠를 자주 소비
- 친구/연인과 함께 여행 계획을 세움
- 구글맵, 네이버 지도, 메모장을 넘나들며 정리하는 것에 피로감
- "이 장소 어디지?" 하며 스크린샷을 다시 찾아보는 경험 있음

## 3. 핵심 가치 제안 (Value Proposition)

### 3.1 SNS 스크린샷 → 장소 자동 추출
```
Before: 스크린샷 저장 → 나중에 검색 → 장소 찾기 → 수동 정리
After:  스크린샷 업로드 → AI가 장소 자동 인식 → 지도에 바로 표시
```

**핵심 기술**: Claude Vision AI로 이미지 내 텍스트/컨텍스트 분석

### 3.2 AI 챗봇 여행 어시스턴트
```
"신주쿠에서 라멘 맛집 추천해줘"
→ AI가 장소 추천 + 이유 설명 + 바로 일정에 추가 가능
```

**핵심 기술**: Google Gemini 2.0 Flash + 스트리밍 응답

### 3.3 통합 여행 플래닝
```
장소 수집 + 지도 시각화 + 일정 편성 + 그룹 공유
→ 하나의 앱에서 모든 여행 계획 완성
```

**핵심 가치**: 여러 앱을 넘나들 필요 없이 한 곳에서 해결

## 4. 핵심 기능

### 4.1 장소 추출 (Place Extraction)
| 입력 방식 | 설명 | 기술 |
|-----------|------|------|
| 이미지 업로드 | SNS 스크린샷에서 장소 추출 | Claude Vision API |
| 텍스트 입력 | 자유 텍스트에서 장소 추출 | Claude Text API |
| URL 입력 | 블로그 URL 크롤링 후 분석 | Cheerio + Claude |
| Google Maps 링크 | 구글맵 URL에서 장소 정보 획득 | Places API |

### 4.2 지도 시각화 (Map Visualization)
- Google Maps에 추출된 장소 마커 표시
- 카테고리별 필터링 (음식점, 카페, 관광지, 쇼핑 등)
- 장소 클릭 시 상세 정보 패널 표시
- 일정 Day별 지도 필터링

### 4.3 일정 관리 (Itinerary Management)
- 날짜 범위 선택 → Day별 일정 자동 생성
- 드래그앤드롭으로 순서 변경, Day간 이동
- 항공편/숙소 정보 관리
- 시간대 설정 가능

### 4.4 AI 챗봇 (Chat Assistant)
- 자연어로 장소 추천 요청
- 추천 장소를 바로 일정에 추가
- 여행 관련 질문 응답
- 대화 맥락 유지

### 4.5 그룹 협업 (Collaboration)
- 초대 링크로 멤버 추가
- 소유자/멤버 역할 구분
- 실시간 동기화 (Supabase Realtime)

### 4.6 공유 (Sharing)
- 읽기 전용 공유 링크 생성
- 공유 페이지에서 프로젝트 복제 가능

- ❌ 과도하게 화려하거나 장식적인 디자인
- ❌ 어린아이 같은 귀여운 톤
- ❌ 복잡하고 기능이 많아 보이는 UI
