## 프로젝트 요약

### 한 줄 소개
> SNS에서 발견한 장소를 실제 여행으로 연결하는 AI 기반 여행 플래너

### 타겟 사용자
- 20-30대 MZ세대
- SNS에서 여행 정보를 수집하는 사용자
- 친구들과 함께 여행을 계획하는 그룹

### 핵심 기능
1. **SNS 스크린샷 → 장소 추출** (AI Vision)
2. **AI 챗봇 여행 어시스턴트** (장소 추천)
3. **지도 시각화** (Google Maps)
4. **드래그앤드롭 일정 관리**
5. **그룹 협업** (초대 링크)

### 브랜드 톤
- ✅ 신뢰감, 전문적
- ✅ 미니멀, 깨끗함
- ❌ 과도한 장식
- ❌ 유치한 톤

## 디자인 요청 사항

### 1차 요청 (전체 리디자인)
1. **브랜드 아이덴티티**
   - 로고 디자인
   - 컬러 팔레트
   - 타이포그래피

2. **디자인 시스템**
   - 컴포넌트 스타일 가이드
   - 아이콘 스타일
   - 스페이싱/그리드 시스템

3. **주요 화면 디자인**
   - 랜딩 페이지
   - 대시보드
   - 프로젝트 상세 (장소/일정 탭)
   - AI 챗봇
   - 모바일 뷰

### 디자인 우선순위
| 우선순위 | 영역 |
|:--------:|------|
| 1 | 전체 시각적 일관성 |
| 2 | 사용성 (UX) 개선 |
| 3 | 여행 플래너로서의 차별화된 아이덴티티 |
