## 2. 촬영 가이드

### 2.1 뷰포트 크기

| 디바이스 | 너비 | 높이 |
|----------|------|------|
| Desktop | 1440px | 900px |
| Tablet | 768px | 1024px |
| Mobile | 375px | 812px (iPhone 13) |

### 2.2 브라우저 설정

1. **Chrome DevTools** 사용 권장
2. Device Toolbar (Ctrl+Shift+M) 활성화
3. 화면 비율: 100% (확대 없음)
4. 브라우저 UI 숨김 (전체 화면 캡처)

### 2.3 촬영 팁

- **일관된 데이터**: 테스트 프로젝트 "도쿄 여행"으로 통일
- **충분한 콘텐츠**: 장소 5개 이상, 일정 3박 4일
- **다양한 카테고리**: 음식점, 카페, 관광지 혼합
- **호버/액티브 상태**: 인터랙션 상태도 캡처
- **모달/패널**: 열린 상태로 캡처

### 2.4 스크린샷 도구

- **macOS**: Cmd + Shift + 4
- **Windows**: Snipping Tool 또는 Win + Shift + S
- **Chrome 확장**: Full Page Screen Capture
- **DevTools**: Capture full size screenshot (Cmd + Shift + P → "screenshot")

## 5. 현재 컴포넌트 목록 (참고용)

디자이너가 현재 사용 중인 UI 컴포넌트를 파악할 수 있도록 목록을 제공합니다.

### shadcn/ui 기반 컴포넌트

| 컴포넌트 | 용도 |
|----------|------|
| Button | 모든 버튼 (variant: default, outline, ghost, destructive) |
| Card | 프로젝트 카드, 장소 카드 |
| Dialog | 모달 (생성, 수정, 삭제 확인) |
| Sheet | 바텀 시트 (모바일 장소 상세) |
| Input | 텍스트 입력 필드 |
| Select | 드롭다운 선택 |
| Switch | 토글 스위치 |
| Tooltip | 툴팁 |
| Avatar | 프로필 아바타 |
| DropdownMenu | 드롭다운 메뉴 |
| Collapsible | 접기/펼치기 영역 |
| Sonner (Toast) | 알림 토스트 |
