# Jira 티켓 코드 리뷰

이 명령어는 특정 Jira 티켓과 연결된 PR을 리뷰합니다.

## 입력
$ARGUMENTS (Jira 티켓 ID 또는 PR 번호)

## 실행 단계

### 1. 관련 정보 조회
- Jira 티켓 정보 (요구사항, AC)
- 연결된 PR 정보
- 변경된 파일 목록

### 2. 코드 리뷰 수행
체크 항목:
- [ ] 요구사항 충족 여부
- [ ] 코드 품질 및 가독성
- [ ] TypeScript 타입 안전성
- [ ] 보안 취약점
- [ ] 성능 이슈
- [ ] 테스트 커버리지

### 3. 리뷰 코멘트 작성
PR에 리뷰 결과 코멘트:
```markdown
## 🔍 코드 리뷰 결과

### ✅ 잘된 점
- 포인트 1
- 포인트 2

### ⚠️ 개선 필요
- [파일:라인] 개선 제안

### 📋 요구사항 체크
- [x] AC 1
- [x] AC 2
- [ ] AC 3 (미충족)

### 💬 추가 의견
의견 내용
```

### 4. Jira 코멘트
티켓에 리뷰 진행 상황 업데이트

## 출력
- 리뷰 요약
- 발견된 이슈 목록
- 승인/수정요청 여부

## 사용 예시
```
/project:jira-review TRAVEL-123
```
